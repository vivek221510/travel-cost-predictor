# ✈️ Travel Cost Predictor

A machine learning-based web app that predicts travel package cost based on user inputs.

## 🔥 Features

* Predicts trip cost using Random Forest
* Considers distance, days, people, transport, hotel, season
* Provides explanation for predictions
* Interactive UI using Streamlit

## 🧠 Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Example Inputs

* Source: Hyderabad
* Destination: Bangalore
* Days: 2
* People: 2

## 💡 Output

Estimated travel cost with explanation.
